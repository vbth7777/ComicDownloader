# ComicDownloader
### Require:
- WebView2
- .Net Core 3.1

### Features:
- Add css selectors of website to download<br/>
- Edit css selectors of website<br/>
- 3 downloading type to choose: Comics, comic, chapter<br/>
- Get css selectors by mouse click with WebView2 (Selectors Setter)
